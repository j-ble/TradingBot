# Coinbase Advanced Trade API

## Welcome to Coinbase Advanced Trade API

## Introduction

The Advanced Trade API enables developers to build trading applications. The platform offers:

- **REST API** for programmatic trading and order management
- **WebSocket protocol** for accessing real-time market data

Review the documentation and guides available before beginning development work.

**Note:** For comprehensive API documentation, endpoint specifications, authentication methods, and code examples, access the full documentation at the Coinbase Developer Platform.

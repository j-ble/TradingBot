# Coinbase Exchange API Overview

## Main Purpose

The platform provides Exchange Trading APIs that allow institutions to place orders and access account information through REST endpoints designed for institutional trading needs.

## Key Documentation Areas

The introductory page directs users to essential resources:

- **Authentication guidance** - securing API access
- **Rate limiting information** - understanding usage constraints
- **Pagination details** - handling large datasets
- **Status code reference** - interpreting API responses
- **Getting started tutorial** - initial implementation steps

## Additional API Options

Beyond the primary REST architecture, the platform supports two alternative communication protocols:

1. **FIX API** - connectivity and reference materials for the Financial Information eXchange protocol
2. **WebSocket API** - real-time feed capabilities and related documentation

## Content Summary

This landing page serves as a navigation hub for developers integrating with Coinbase's institutional trading infrastructure, consolidating links to authentication, technical specifications, and multiple API implementations.
